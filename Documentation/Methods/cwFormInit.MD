## Description
Initialise les formulaires html.

```4d
cwFormInit ( $formName_t ) -> $codeHtml_t
```

| Paramètres  | Type  | entrée/sortie | Description |
| ----------- | ----- | ------------- | ----------- |
| $formName_t | Texte | entrée        | Nom du formulaire html |
| $codeHtml_t | Texte | sortie        | Code html du formulaire |
