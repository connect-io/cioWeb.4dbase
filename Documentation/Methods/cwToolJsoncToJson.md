## Description
Converti du JSONC en JSON

```4d
cwToolJsoncToJson ( $textJsonc_t ) -> $textJson_t
```

| Paramètres       | Type    | entrée/sortie | Description |
| ---------------- | ------- | ------------- | ----------- |
| $textJsonc_t     | Texte   | entrée        | Texte en format JSONC |
| $textJson_t      | Texte   | sortie        | Texte en format JSON  |
