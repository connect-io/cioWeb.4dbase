## Description
Reconstruire une date depuis une saisie client.

```4d
cwDateClean ( $clientInput_t ) -> $result_t
```

| Paramètres     | Type  | entrée/sortie | Description |
| -------------- | ----- | ------------- | ----------- |
| $clientInput_t | Texte | entrée        | Saisie client |
| $result_t      | Texte | sortie        | Date formatté |
